# 가운데 정렬 레이아웃 안정화 계획

본 계획은 가운데 정렬 모드에서 애니메이션 적용 시 발생하는 레이아웃 틀어짐(박스 크기 축소 등)을 해결하기 위한 절차를 담고 있습니다.

## 1. 접근 방식
*   애니메이션 래퍼 요소인 `.slide-wrapper`에 `display: contents`를 적용하여, 레이아웃 구조를 깨뜨리지 않고 기존의 Flexbox 정렬 로직이 자식 요소들에게 직접 전달되도록 합니다.

---

## 2. 세부 구현 단계

### 단계 1: 스타일 시트 업데이트 (`src/styles/slide.css`)
*   `.slide-wrapper` 클래스에 `display: contents;` 스타일 추가.

### 단계 2: 에디터 로직 검증 (`src/editor.js`)
*   이미 적용된 빈 태그 제거 로직이 정상 작동하는지 재점검.
