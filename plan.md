# PDF 내보내기 기능 구현 계획

본 계획은 기존의 마크다운 내보내기 기능을 확장하여, 슬라이드 디자인이 그대로 유지된 PDF 파일 저장 기능을 추가하기 위한 절차를 담고 있습니다.

## 1. 접근 방식
1.  **드롭다운 UI 도입**: "내보내기" 버튼을 클릭하면 마크다운(.md)과 PDF(.pdf)를 선택할 수 있는 메뉴를 제공합니다.
2.  **브라우저 프린트 시스템 활용**: 
    *   `window.print()`와 `@media print` CSS를 사용하여 별도의 라이브러리 없이 고해상도 PDF 출력을 구현합니다.
    *   출력 전용 컨테이너(`#print-container`)에 모든 슬라이드를 렌더링하고, 테마 및 레이아웃 설정을 그대로 반영합니다.

---

## 2. 세부 구현 단계

### 단계 1: UI 업데이트 (`index.html`)
*   `btn-export`를 드롭다운 트리거로 변경.
*   마크다운 및 PDF 내보내기 버튼이 포함된 `export-menu` 추가.
*   출력 전용 컨테이너 `#print-container` 추가.

### 단계 2: 스타일링 (`src/styles/editor.css`)
*   내보내기 드롭다운 메뉴 스타일링.
*   `@media print` 쿼리를 작성하여 출력 시 UI를 숨기고 슬라이드만 16:9 비율로 페이지별 출력되도록 설정.

### 단계 3: 내보내기 로직 구현 (`src/main.js`)
*   드롭다운 토글 기능 구현.
*   PDF 내보내기 시:
    *   현재 에디터의 슬라이드 데이터를 `#print-container`에 복제.
    *   현재 테마 및 레이아웃 상태를 반영.
    *   `window.print()` 호출 후 임시 데이터 정리.
