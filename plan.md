# PDF 내보내기 고도화 계획 (라이브러리 도입)

본 계획은 브라우저 인쇄 기능의 한계를 극복하고, 슬라이드 디자인을 완벽하게 보존하는 PDF 내보내기 기능을 구현하기 위한 절차를 담고 있습니다.

## 1. 접근 방식
*   `html2pdf.js` 라이브러리를 사용하여 DOM을 이미지화한 후 PDF로 변환하는 방식을 채택합니다.
*   인쇄 대화상자를 거치지 않고 버튼 클릭 시 즉시 PDF 파일이 생성 및 다운로드되도록 합니다.

---

## 2. 세부 구현 단계

### 단계 1: 라이브러리 추가 (`index.html`)
*   `html2pdf.bundle.min.js` CDN 추가.

### 단계 2: 내보내기 로직 전면 수정 (`src/main.js`)
*   `window.print()` 호출 대신 `html2pdf()` API 호출로 변경.
*   고해상도 캡처를 위한 `html2canvas` 옵션 설정.
*   슬라이드별 페이지 분할 옵션(`pagebreak`) 적용.

### 단계 3: 스타일 보정 (`src/styles/editor.css`)
*   프린트용 스타일 대신 캡처용 임시 컨테이너 스타일로 변경.
