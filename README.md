# MD Slide 🎬

마크다운(Markdown)을 작성하면 실시간으로 미니멀하고 아름다운 프레젠테이션 슬라이드로 변환해주는 웹 애플리케이션입니다. 유튜버나 발표자를 위해 빠르고 직관적으로 슬라이드를 구성할 수 있도록 돕습니다.

## 🚀 프로젝트 개요

- **실시간 렌더링**: 에디터에 마크다운을 입력하면 우측 화면에 즉시 슬라이드 형태의 프리뷰가 생성됩니다.
- **다양한 테마와 레이아웃**: Midnight, Paper, Terminal, Abyss 등의 테마를 지원하며, 슬라이드의 기본 정렬(좌상단/가운데)을 쉽게 변경할 수 있습니다.
- **특수 매크로 지원**: 일반 마크다운 문법 외에도 주석(HTML Comment) 형태의 매크로를 제공하여 제목 슬라이드, 목차, 레이아웃 분할, 애니메이션(프래그먼트) 효과 등을 쉽게 적용할 수 있습니다.
- **웹 기반 프레젠터**: 별도 프로그램 없이 브라우저에서 '슬라이드쇼' 버튼을 누르면 OS 레벨의 전체화면(Fullscreen) 모드로 발표를 진행할 수 있습니다.
- **데이터 관리**: 작성한 슬라이드를 로컬 환경으로 내보내거나(Export) 가져오기(Import)를 지원합니다.

## ✨ 주요 매크로 (명령어)

슬라이드 텍스트 내에 아래와 같이 HTML 주석 형태로 입력하여 특수 기능을 활성화합니다.

*   `<!-- cover -->`: **제목(Cover) 슬라이드**. 텍스트를 크게 중앙에 배치하여 표지처럼 만듭니다.
*   `<!-- toc -->`: **세로 목차(TOC) 슬라이드**. 리스트 요소 번호와 선을 스타일링하여 세로형 목차를 만듭니다.
*   `<!-- toc-h -->`: **가로 목차(TOC) 슬라이드**. 리스트 요소를 카드 형태로 가로로 나열합니다.
*   `<!-- animate -->`: **애니메이션(프래그먼트) 효과**. 슬라이드 내의 요소(리스트, 이미지 등)를 한 번에 보여주지 않고 클릭할 때마다 순차적으로 나타나게 합니다.
*   `<!-- split-left -->` / `<!-- split-right -->`: **이미지 분할 레이아웃**. 텍스트와 이미지를 좌/우로 2단 분할하여 배치합니다.

## 🛠️ 실행 방법

본 프로젝트는 [Vite](https://vitejs.dev/) 기반으로 구축되었습니다. 실행을 위해 **Node.js**가 설치되어 있어야 합니다.

1. **의존성 설치**
   프로젝트 루트 폴더(package.json이 있는 위치)에서 아래 명령어를 실행하여 패키지를 설치합니다.
   ```bash
   npm install
   ```

2. **개발 서버 실행**
   개발 모드로 로컬 서버를 실행합니다.
   ```bash
   npm run dev
   ```
   실행 후 터미널에 나타나는 주소(일반적으로 `http://localhost:3000/` 또는 `http://localhost:5173/`)로 브라우저에서 접속합니다.

3. **프로덕션 빌드 (선택 사항)**
   서비스를 배포하기 위해 정적 파일을 생성하려면 아래 명령어를 사용합니다.
   ```bash
   npm run build
   ```
   빌드된 파일은 `dist` 폴더에 생성됩니다.

## ⌨️ 단축키 안내

*   `Ctrl + Enter`: 슬라이드쇼 시작
*   `←`, `→` 또는 `클릭` (화면 좌/우 분할 기준): 이전 / 다음 슬라이드 이동
*   `F`: 전체화면 토글
*   `Escape`: 슬라이드쇼 종료
*   `Ctrl + S`: 마크다운 파일(.md) 내보내기 저장
