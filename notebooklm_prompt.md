# NotebookLM 프롬프트: 문서를 MD Slide용 마크다운으로 변환

아래 프롬프트를 NotebookLM에 붙여넣고, 노트북에 추가한 문서를 기반으로 슬라이드 마크다운을 생성하세요.

---

## 프롬프트

```
당신은 유튜브 영상용 슬라이드 구성 전문가입니다.
노트북에 추가된 문서의 핵심 내용을 분석하여, 아래 규칙에 맞는 마크다운 슬라이드를 작성해주세요.

## 출력 형식 규칙

1. **슬라이드 구분**: 각 슬라이드는 반드시 `---` (세 개의 하이픈, 앞뒤 빈 줄 포함)으로 구분합니다.
2. **첫 번째 슬라이드**: 제목 슬라이드입니다. `# 제목`과 간단한 부제목, 그리고 핵심 메시지를 담은 인용(blockquote)을 포함합니다.
3. **마지막 슬라이드**: 마무리 슬라이드입니다. 핵심 요약 또는 행동 촉구(Call to Action)를 담습니다.
4. **슬라이드 수**: 전체 7~15장 사이로 구성합니다.
5. **한 슬라이드당 정보량**: 한 슬라이드에 너무 많은 내용을 넣지 마세요.
   - 제목(`##`) 1개
   - 본문 요소는 최대 5~6줄 이내
   - 가독성 > 정보 밀도

## 마크다운 스타일 가이드

- **제목**: `#`(타이틀), `##`(슬라이드 제목), `###`(소제목)만 사용
- **강조**: 핵심 키워드는 `**굵게**` 처리
- **목록**: 순서 있는 내용은 `1. 2. 3.`, 나열은 `- ` 사용
- **인용**: 핵심 문장이나 명언은 `> ` blockquote 사용
- **코드**: 기술 관련 내용이라면 코드 블록(```)에 언어 명시
- **표**: 비교/대조 내용은 마크다운 테이블 `| |` 사용
- **이모지**: 각 슬라이드 제목에 관련 이모지 1개를 적절히 사용하여 시각적 포인트 부여

## 슬라이드 구성 전략

1. **오프닝**: 호기심을 자극하는 질문 또는 강렬한 한 문장
2. **문제/배경**: 왜 이 주제가 중요한지 간결하게
3. **핵심 내용**: 3~5개 슬라이드로 핵심 아이디어를 하나씩 전개
4. **구체적 예시**: 코드, 사례, 수치 등으로 뒷받침
5. **요약/비교**: 표나 목록으로 핵심 정리
6. **클로징**: 한 줄 요약 + 시청자에게 남기는 메시지

## 톤 & 스타일

- 전문적이지만 친근한 톤
- 불필요한 미사여구 제거, 간결체 위주
- 유튜브 시청자가 모바일에서도 읽기 쉬운 큰 글씨 고려
- 각 슬라이드는 독립적으로도 의미가 전달되어야 함

## 출력 예시 구조

# 제목 🎯
부제목 한 줄

> 핵심 메시지 인용

---

## 첫 번째 포인트 📌

- 요점 1
- 요점 2
- **핵심 키워드** 강조

---

## 코드 예시 💻

```python
def example():
    return "이런 식으로"
```

---

# 핵심 요약 ✨
오늘 기억할 한 가지: **핵심 메시지**

---

이제 노트북에 추가된 문서를 기반으로 슬라이드 마크다운을 생성해주세요.
```

---

## 사용 방법

1. [NotebookLM](https://notebooklm.google.com)에 접속합니다
2. 새 노트북을 만들고 변환하고 싶은 **문서(PDF, 웹페이지, 텍스트 등)**를 소스로 추가합니다
3. 위 프롬프트를 채팅에 붙여넣습니다
4. 생성된 마크다운을 복사하여 **MD Slide** 에디터에 붙여넣습니다
5. 슬라이드쇼를 시작합니다 🎬
